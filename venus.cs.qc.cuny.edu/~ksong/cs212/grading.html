Project 4 Instructions | project4.zip

  • Bring in (copy and paste) the Card class from Lab 4. Adjust import statements if necessary.
  • Use the built-in LinkedList class from the java.util package (LinkedList API). Do not use the LinkedList class from our lab. Feel free to use any of the methods offered in the LinkedList API (look into: add, addAll, get methods). You may also be interested in the following built-in classes (but not required to use all):
    • Collections API (sort method)
    • Collection API (removeAll method)
    • Scanner API (Reading input from console)
    • ArrayList API
  • You are not required to use an Iterator; you may use an enhanced FOR loop to access each element in your linked list. Example
  • Hand class: The findRank and getCards method have the same method description but should have different behaviors. The findRank method should search for cards of the specified rank and return a linkedlist of those cards. The getCards method is called when the player successfully fishes a rank from the opponent's hand. It should remove the cards of the same rank from the deck and return a linkedlist of those cards.
  • The player should fish for the opponent's card using a keyboard entry. The computer should randomly pick a rank from one of the cards in its hand. Both players can only fish for ranks in their hands.
  • Shuffle the cards to get random draws for every run.
